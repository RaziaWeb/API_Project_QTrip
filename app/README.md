# QTrip API Automation

## 📌 Project Description

This project automates REST APIs of QTrip application using RestAssured and TestNG.

## 🚀 Tech Stack

* Java
* RestAssured
* TestNG
* Gradle
* Apache POI (Excel - Data Driven Testing)

## ✅ Features Implemented

* User Registration API
* Login API
* Cities API
* Adventures API
* Reservation APIs (Create, Get, Delete)
* Data-driven testing using Excel
* Reusable DataProvider using TestNG

## 📂 Project Structure

* tests → API test cases
* utils → Utility classes
* resources → Test data (Excel)

## ▶️ How to Run

./gradlew test
